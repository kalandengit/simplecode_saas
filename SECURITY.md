# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly:

1. **Email:** kalan.kalandeh@gmail.com
2. **Subject:** `[SECURITY] simplecode_saas — <short description>`
3. Do **not** open a public GitHub issue for security vulnerabilities.

## Response Times

- **Acknowledgment:** within 48 hours
- **Initial assessment:** within 5 business days
- **Fix or mitigation ETA:** communicated after assessment, depending on severity

## Disclosure

Please allow a reasonable time for remediation before any public disclosure.
We commit to crediting reporters (unless they prefer to remain anonymous).
